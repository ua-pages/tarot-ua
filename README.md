# Tarot Atlas (навчальний проєкт для портфоліо)

Повноцінний застосунок про карти Таро українською мовою.

**Технології:**
- Клієнт: чистий JS (веб-компоненти), односторінковий застосунок
- Сервер: Node.js (чистий HTTP) — статичний файловий сервер
- ШІ-тлумачення: тлумач на основі правил (rule-based), інтегрований у клієнт
- Сховище: IndexedDB + localStorage (журнал розкладів, обране, поширені розклади)

## Можливості

- **Карта дня** — визначена карта на кожен день
- **Розклади:** 3 карти («Минуле·Теперішнє·Майбутнє»), Пентаграма балансу (5 карт), Кохання та стосунки (5), Кар'єра та гроші (5)
- **Тлумачення** — цілісний аналіз розкладу з трьома тонами: психологічний, містичний, практичний
- **Бібліотека карт** — 78 карт з ілюстраціями, значеннями й ключовими словами
- **Журнал розкладів** — збереження, нотатки, обране в IndexedDB
- **Профіль** — локальне ім'я (без реєстрації)
- **Публічні розклади** — поширення за посиланням, зберігаються локально в IndexedDB
- **SEO** — окремі сторінки для кожної карти та розкладу (SSR-подібна генерація на клієнті)

## GitHub Pages

Сайт доступний за адресою: [https://m00rlock.github.io/tarot-ua/](https://m00rlock.github.io/tarot-ua/)

Деплой відбувається автоматично через GitHub Actions при пуші в гілку `main`.

### Локальна розробка

```bash
node apps/web/server.js  # http://localhost:5173/tarot-ua/
```

Сервер автоматично стріпає префікс `/tarot-ua` з URL, тому всі шляхи в коді працюють як на GitHub Pages.

## Структура

```
apps/web/
├── public/
│   ├── css/              # Стилі
│   ├── cards/            # SVG-ілюстрації карт (78 шт.)
│   ├── seo/              # OG-зображення
│   ├── js/
│   │   ├── components/   # Веб-компоненти (18 шт.)
│   │   ├── services/     # tarot-engine, api, session-storage, journal-storage, indexed-db
│   │   ├── analytics/    # PostHog аналітика
│   │   ├── seo/          # SEO-мета
│   │   ├── constants/    # Конфігурація
│   │   └── lib/          # Сторонні бібліотеки (karbovanets)
│   ├── sitemap.xml       # 78 карт + реверси + розклади
│   └── robots.txt
├── server.js             # Статичний файловий сервер
└── package.json
```

## Швидкий старт

```bash
# 1. Встановити залежності
npm install

# 2. Запустити сервер
node apps/web/server.js
```

Жодна БД або API не потрібна. Застосунок працює повністю автономно.

## Аналітика

Аналітика за бажанням через PostHog. Файл `apps/web/.env`:

```env
POSTHOG_KEY=phc_...
POSTHOG_HOST=https://eu.i.posthog.com
```

Для вимкнення: `localStorage['tarot-analytics-opt-out'] = 'true'`.
